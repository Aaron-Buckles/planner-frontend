# planner-frontend

## Project setup

```
npm install
```

Create `.env` file in the root folder like:

```
VUE_APP_API_BASE_URL=http://127.0.0.1:5000/api
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
